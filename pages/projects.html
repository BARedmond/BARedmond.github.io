Zip Code Map

My job as a package handler at Fedex Ground required me to scan each package and sort it according to its destination trailer. It is fast paced, and can be made more efficient if scanning is eliminated. Over time, I familiarized myself with every trailer destination and the zip codes of the surrounding areas. As a result, I was able to correctly sort over 95% of the packages that came through entirely without a scanner. I was dedicated to helping others learn zipcodes as well, and I set out to find the best way to teach. Using photoshop, I created this map that shows a visual representation of the area's destinations and the zipcodes that went to that trailer.

"Tracker"

I started with a 7 pocket accordian binder... and swiftly upgraded to the 13 pocket. I still needed a way to organize my thoughts, however. I figured I'd end up changing more, but the design has worked very well for me.

Future Home

My eventual goal is to own a home without locking myself into a mortgage. I would love to build my own home, but doing this without a loan means the home would have to be quite small. My home should start off small, but as I accumulate more money, the home should be easily added onto. It requires me to decide what is truly necessary and to design with renovations in mind.
I started out by jumping straight into the project using Chief Architect. It was easy to figure out how to do things like create rooms and add dormers to a gable roof, but I realized I didn't really "know" the program. I then decided that Revit would be a better program to learn and that I should actually LEARN IT before jumping straight into my project. That's where I am at right now; teaching myself good practices, such as schematic design, levels, scheduling, and drawing sets.
